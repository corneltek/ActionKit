Internet Explorer support for CSS3 border-radius and box-shadow

Thanks to Remiz Rahnas for getting this script started, we now have basic support for two of the most useful CSS3 styles in Internet Explorer. It should work in IE6, IE7, and IE8 in both Quirks Mode and Standards Mode.

Few notes:

  • border-radius can only set all four corners at once. This is a limitation of Microsoft's VML.
  • box-shadow ignores the color you specify and always uses black instead. Also a VML limitation.
  • The script works by adding new elements (VML) at the same position as the target element but with a lower z-index. It won't break your layout, but you may need to pay attention to the z-indexes. See here for a view of what this page's DOM looks like in IE.
